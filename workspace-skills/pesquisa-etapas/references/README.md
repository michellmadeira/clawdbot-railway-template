# Prompts por etapa (papéis)

Coloque aqui um arquivo `.md` por etapa, com nome numérico para definir a ordem:

- `01-<papel>.md` — primeira etapa (ex.: coletor)
- `02-<papel>.md` — segunda etapa (ex.: revisor)
- `03-<papel>.md` — terceira etapa (ex.: sintetizador)
- …

O conteúdo de cada arquivo é o **prompt/instrução** que o agente deve seguir naquele papel. O agente lê estes arquivos em ordem e executa cada um como uma etapa da pesquisa.

Os arquivos `01-exemplo-coletor.md`, `02-exemplo-revisor.md` e `03-exemplo-sintetizador.md` são placeholders. Substitua pelo seu conjunto de prompts ou renomeie/edite conforme sua sequência.
